# Jeu de la Vie

Ce projet implémente une version du Jeu de la Vie de Conway en Java.

## Prérequis

- Java 25
- Maven

## Compilation

Compiler le projet :

```sh
mvn compile
```

Nettoyer les fichiers :

```sh
mvn clean
```

## Exécution

Pour exécuter l'application :

```sh
mvn exec:java
```

## Structure

- `src/main/java` : code source
